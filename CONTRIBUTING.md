# 🤝 Contribuições (Read-Only)

Obrigado pelo seu interesse neste repositório! 

Este projeto funciona como um **Portfólio Pessoal de Estudos Cloud**. Sendo assim, **não estou aceitando solicitações de Pull Requests (PRs) ou contribuições abertas de código no momento**. Toda a evolução, documentação e infraestrutura provisionada aqui são de autoria pessoal para demonstrar conhecimento técnico e arquitetural na AWS.

## Como utilizar este repositório

- 🔍 Sinta-se à vontade para ler, aprender com os laboratórios e inspecionar a infraestrutura como código (`iac/`).
- 🍴 Fique à vontade para fazer um **Fork** deste repositório para o seu próprio GitHub caso queira replicar os cenários de estudo e praticar no seu ambiente.
- 💬 Se encontrar algum erro grave de conceito ou tiver uma sugestão construtiva, você pode abrir uma [Issue](https://github.com/caiocesarti/AWS-Labs/issues) para compartilhar seu feedback.

Obrigado por apoiar a comunidade de tecnologia e bons estudos! ☁️
